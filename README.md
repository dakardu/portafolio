# Portafolio Personal

Portafolio profesional desarrollado con React, Node.js y MongoDB Atlas, completamente contenerizado con Docker.

## Tecnologías

### Frontend

- React
- Vite
- Axios
- React Router

### Backend

- Node.js
- Express
- MongoDB Atlas
- Mongoose

### Infraestructura

- Docker
- Docker Compose
- NGINX Reverse Proxy

## Arquitectura

```text
Frontend (React)
       |
       v
Backend (Express API)
       |
       v
MongoDB Atlas
```

## Ejecución local

```bash
docker compose up -d --build
```

Frontend:

```text
http://localhost:5173
```

API:

```text
http://localhost:3000/api/projects
```

## Funcionalidades

- Presentación profesional
- Información personal y experiencia
- API REST para proyectos
- Integración con MongoDB Atlas
- Despliegue mediante Docker
